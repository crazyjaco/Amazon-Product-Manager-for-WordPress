== Description ==
Query and store Amazon.com products as custom post types. Display in widgets.

== Installation ==
1. Upload "/Amazon-Product-Manager-for-WordPress" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
